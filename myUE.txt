My Universal Extractor:
-----------------------
MyUE is a command line program that can be used to extract the contents out of a lot of archive types and installer packages.

Readme:
-------
My Universal Extractor is inspired by Universal Extractor from Jared Breland (http://www.legroom.net/software/uniextract).
This is a simplified (and updated) command line version.

Installation is simple: start myUE.exe as administrator and follow the instructions.
MyUE will then write one (or two) registry key(s) so that a menu item is added to every context menu when right clicking a file (or directory).

The program can (of course) be run from the command line. For the syntaxis: run the program without parameters.

When you select "Process with myUE", myUE will determine the file type and extract its contents to a subfolder or give other options.
